# Troubleshooting Blank Page

## Common Causes

1. **JavaScript Errors** - Check browser console (F12)
2. **Missing Dependencies** - Run `npm install`
3. **Import Errors** - Check file paths
4. **React Context Errors** - Verify ThemeContext setup

## Quick Fixes

1. Open browser DevTools (F12 or Cmd+Option+I)
2. Check Console tab for red error messages
3. Share the error messages for debugging

## Manual Check

Open browser console and look for:
- Import errors
- "ThemeContext" or "ThemeProvider" errors
- Missing module errors
- React render errors
